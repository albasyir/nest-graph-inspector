## Install Package

Add Nest Graph Inspector to your NestJS project.

::code-group
```bash [npm]
npm install nest-graph-inspector
```

```bash [yarn]
yarn add nest-graph-inspector
```

```bash [pnpm]
pnpm add nest-graph-inspector
```

```bash [bun]
bun add nest-graph-inspector
```
::

## Install Module

Import `NestGraphInspectorModule` in your root module. The minimum setup uses the Viewer output, which installs a graph endpoint and prints a message in your application console when the app starts.

```ts [app.module.ts]
import { Module } from '@nestjs/common';
import { NestGraphInspectorModule } from 'nest-graph-inspector';

@Module({
  imports: [
    NestGraphInspectorModule // that's it!
  ],
})
export class AppModule {}
```

## Run Project

```bash
npm run start
```

After startup, the inspector logs the Viewer URL and graph endpoint path in your console. Open the Viewer, enter your app origin, and inspect the live dependency graph.
