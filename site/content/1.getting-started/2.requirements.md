## Requirements

Nest Graph Inspector is built for modern NestJS applications and reads from the runtime Nest container after your app has initialized.

| Requirement | Version |
| ----------- | ------- |
| NestJS | 10 - 11 |

::callout{icon="i-lucide-lightbulb" color="neutral"}
Earlier NestJS versions may still work, but they are not officially supported. You can try force installing:

::package-manager-command{command="force-install"}
::

If it works for your project, [raise an issue](https://github.com/albasyir/nest-graph-inspector/issues) so support coverage can be updated.
::
