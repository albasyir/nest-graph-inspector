## output.file

The `output.file` option specifies the filename for the generated graph output.

```ts
output: {
  file: 'graph.md'
}
```

### Markdown Output

When the file has a `.md` extension, the output includes:

- A **module list** with all loaded modules
- **Import relationships** between modules
- **Exported providers** from each module
- **Providers with dependencies** listed per module
- **Controllers with dependencies** listed per module
- **Mermaid diagrams** showing the dependency graph visually

::callout{icon="i-lucide-lightbulb" color="primary"}
Mermaid diagrams render natively on GitHub. For local viewing, use a Mermaid-compatible Markdown viewer or install a VSCode extension.
::

### JSON Output

When the file has a `.json` extension, you get structured data suitable for programmatic analysis or custom tooling.

### Examples

```ts
// Markdown with Mermaid diagrams
output: { file: 'docs/architecture.md' }

// JSON for tooling
output: { file: 'reports/deps.json' }
```
